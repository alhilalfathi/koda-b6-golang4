## Celcius Convertion Program
Program ini digunakan untuk melakukan konversi dari celcius ke fahrenheit, reamur, atau kelvin.
Dibuat menggunakan package fmt untuk menghandle input output, lalu convert function dipisahkan ke module convertion kemudian akan dipanggil pada function main untuk melakukan konversi suhu.

# Cara menggunakan
1. Download semua resource di repo ini
2. jalankan main.go
3. masukan nilai suhu celcius
4. pilih target konversi suhu
